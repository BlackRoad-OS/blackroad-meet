# BlackRoad Meet

Self-hosted video conferencing with end-to-end encryption.

## Features
- **HD Video** - High-quality video calls
- **E2E Encryption** - Secure communications
- **Screen Sharing** - Share screens and apps
- **Recording** - Local and cloud recording
- **Breakout Rooms** - Split into smaller groups
- **Self-Hosted** - Full privacy control
